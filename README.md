# DynGRN
DynGRN is a tool for deciphering fine-grained dynamic life processes using time-series transcriptomics data. It takes time-series gene expression profiles (snapshots data or pseudo-time-series data) and cell-type-specific prior regulatory knowledge as inputs, then conducts cell-type-specific GRN construction and dynamic GRNs rewiring.
